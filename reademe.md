# How to setup


### If you're me use:
```
cd ~/.config/Code/User
git init
git remote add origin git@github.com:sirgallifrey/my-vscode-conf.git 
# https://github.com/sirgallifrey/my-vscode-conf.git
git fetch
git reset --hard origin/master
```


### If you're not me use:
```
cd ~/.config/Code/User
git init
git remote add origin git://github.com/sirgallifrey/my-vscode-conf.git
git fetch
git reset --hard origin/master
```
